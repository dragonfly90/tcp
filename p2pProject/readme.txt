1.Environment: python 2.7
2.Start the server: python server.py
3.Start the client:
   run python client.py
   Firstly, you will input the ip address of the server. If the server is running in current client, you can input 0, otherwise you
   should input 1, then input the ipv4 address.
4.The format of RFC file is: rfc1.txt. There is the number between 'rfc' and '.txt'. The rfc files should be in txt format
